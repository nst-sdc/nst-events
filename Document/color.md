## 7. Component-Specific Gradient Recommendations

### Buttons
*   **Primary Button**: Linear Gradient (135deg, `#0052FF` → `#3E3A7E`), Text: `#F7E8A4`
*   **Success Button**: Linear Gradient (135deg, `#4FD1C5` → `#2DB5A0`), Text: `#1C2044`
*   **Warning Button**: Linear Gradient (135deg, `#FFA500` → `#FF7F50`), Text: `#1C2044`

### Cards
*   **Light Card**: Linear Gradient (135deg, `#F7E8A4` → `#EBD0AA`), Border: `1px solid #D9A8BD`
*   **Dark Card**: Linear Gradient (135deg, `#3E3A7E` → `#2F345C`), Border: `1px solid #5B47A4`

### Status Badges
*   **Active/Enrolled**: Linear Gradient (135deg, `#4FD1C5` → `#2DB5A0`)
*   **In Progress**: Linear Gradient (135deg, `#A06BFF` → `#8A5CC2`)
*   **Due Soon**: Linear Gradient (135deg, `#FFA500` → `#FF7F50`)
*   **Overdue**: Linear Gradient (135deg, `#D32F2F` → `#B71C1C`)
